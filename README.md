## Run Locally

Clone the project

```bash
  git clone https://github.com/Iamstanlee/dufuna_app.git
```

Go to the project directory

```bash
  cd dufuna_app
```

Install dependencies

```bash
  flutter pub get
```

Run the app android or ios simulator

```bash
  flutter run
```

## Running Tests

To run tests, run the following command

```bash
  flutter test
```
